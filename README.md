# CoV19-T

## Description
Dash Plotly visualisation of COVID19 cases in Africa based on data from [Our World In Data Github Repository](https://github.com/owid/covid-19-data)  
At present updates are done manually to the [african.csv](https://github.com/skuodi/CoV19-T/blob/master/data/african.csv) file




## Usage

Head over to the [live site](https://tizen-rumbler.herokuapp.com) hosted on Heroku (best viewed on a large screen).  
Use the searchbox to select your country of interest ; or more than one for a comparison.  
Hovering your mouse over the graph shows labelled data corresponding to the date.
